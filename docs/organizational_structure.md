## Email Routing Configuration

**All emails route through:** frontdeskllc@outlook.com

**Email Flow:**
1. External email sent to: `role@frontdeskagents.com`
2. Appears from: `Role Name <role@frontdeskagents.com>`
3. Reply-To header: `frontdeskllc@outlook.com`
4. Actual delivery: via Resend API
5. Replies received at: `frontdeskllc@outlook.com`
6. Internal routing: Based on original sender

## Access Levels

### Level 1: Executive
- Full access to all email addresses
- Can send as any executive or department address
- Access to all distribution lists

### Level 2: Department Head
- Access to department email addresses
- Can send as department role addresses
- Access to department distribution lists

### Level 3: Manager
- Access to team email addresses
- Can send as team role addresses
- Access to team distribution lists

### Level 4: Employee
- Access to personal email address
- Can send as personal address only
- Access to relevant distribution lists

## Maintenance & Updates

### Adding New Employee
1. Create email address: `firstname.lastname@frontdeskagents.com`
2. Add to relevant distribution lists
3. Configure email signature
4. Provide access credentials
5. Update employee directory

### Removing Employee
1. Disable email address
2. Remove from distribution lists
3. Forward emails to manager
4. Archive email data
5. Update employee directory

### Changing Roles
1. Update email aliases
2. Modify distribution list memberships
3. Adjust access levels
4. Update email signature
5. Notify relevant teams
